Highlights

Favoriten, Suche und schnelle Senderwechsel. Unterstützt gängige AAC/MP3-Streams.

VLC Engine

Zuverlässige Wiedergabe dank libVLC – auch für längere Sessions und viele Formate.

Dunkles Theme

Klares, lesbares UI in satten Blautönen. Fokus auf Tastatur & Maus-Usability.

Einfaches Hinzufügen, Entfernen und Sortieren. Fortschritt speichert in AppData.

FAQ

Wie aktiviere ich GitHub Pages?

Repo → Einstellungen → „Pages“ → Branch main und Ordner / oder /docs wählen. Danach ist die Seite unter https://DEIN-USER.github.io/DEIN-REPO erreichbar.

Wie biete ich Downloads an?

Leg die Dateien in /downloads ab und verlinke sie (siehe oben). Für Dateien >100MB nutze GitHub Releases.

Welches Lizenzmodell nutzt BeyondApp?

MIT. Füge eine LICENSE-Datei im Repo hinzu und nenne die Lizenz im Footer.
